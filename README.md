# q30
A simple script to calculate q20/q30 percentages of a fastq file (can be gzipped)
```shell
python q30.py <fastq_file_name>
```
# output
```
('total bases:', 1386659)
('q20 bases:', 1280980)
('q30 bases:', 1232024)
('q20 percents:', 92.37887613320939)
('q30 percents:', 88.84837584438567)
Time used: 0.221389055252
```
